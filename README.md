# RoadEase - نظام إدارة ورش السيارات

## نظرة عامة
برنامج سطح مكتب متكامل لإدارة ورش صيانة السيارات، يجمع بين نظام ERP و CRM، ويدعم اللغتين العربية والإنجليزية.

## الميزات الرئيسية
- واجهة ثنائية اللغة (عربي/إنجليزي) مع إمكانية التبديل بسهولة.
- إدارة العملاء، السيارات، طلبات الصيانة، المخزون، الفواتير، المصروفات، والموظفين.
- نظام صلاحيات متقدم للموظفين.
- الفوترة الضريبية مع طباعة PDF وتصميم الفاتورة مع شعار الورشة.
- رفع شعارات الورشة لاستخدامها في الواجهة والفواتير.
- نظام الولاء للعملاء مع تصدير تقارير Excel.
- إرسال الفواتير عبر البريد الإلكتروني والواتساب.
- لوحة قيادة تعرض نظرة عامة وتقارير مالية.
- تخصيص ألوان الواجهة.
- نسخ احتياطي تلقائي لقاعدة البيانات.

## المتطلبات
- Python 3.11+
- المكتبات: موجودة في ملف requirements.txt
